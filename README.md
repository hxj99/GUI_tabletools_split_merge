# GUI_tabletools_split_merge
本项目这要目的是对excel表格处理，解决表格的拆分和合并。使用python编程并且tkinter图形化界面，应用pandas库处理表格数据。
* 默认打开第1个工作表，也可自行设定；
* 第1行为标题，第2行为数据开始行，结束行可以自行设定；
* 默认取出所有的列，可以自行设定想要取出的列；
* 拆分时，根据自己设定的列内容进行拆分；
* 自己设定输出的保存位置。
